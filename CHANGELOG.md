# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-04-29

Marks the start of the current release line. Requires Node 18 or higher.

### Changed

- Minimum Node version bumped to 18
- Exports corrected for ESM and CJS consumers — `types` now resolves per condition (`index.d.mts` for ESM, `index.d.cts` for CJS)

### Notes

- For Node 16 / Angular 12 support, use `npm install name-capitalize@legacy` (v1 branch).

---

## [1.1.0] - 2026-04-29

Last release of the legacy line before the v1/v2 split.

### Changed

- Promoted to `legacy` npm tag — install with `npm install name-capitalize@legacy`

---

## [1.0.2] - 2026-04-29

### Fixed

- Corrected `exports` field in `package.json` to match the output generated by `tsdown` (`.cjs` / `.mjs` / `.d.cts` / `.d.mts`)

---

## [1.0.1] - 2026-04-29

### Added

- Legacy support for Node 16 and Angular 12

---

## [1.0.0] - 2026-04-28

Initial release.

### Added

- `capitalizeName` function with support for particles (`de`, `van`, `von`, `di`, `bin`…)
- Hyphenated compound names (`Jean-Pierre`)
- Apostrophe prefixes (`O'Higgins`, `D'Artagnan`)
- Full Unicode support (`Ñ`, `Ö`, `Ş`, `Å`…)
- Zero dependencies
